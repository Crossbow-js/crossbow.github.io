Beyond NPM scripts

Unifying the use & configuration of plain functions, gulp-style streams, tasks in separate files, shell/NPM scripts plus much more!

Get Started!

What's different?

We are NOT creating a whole new eco-system of plugins, rather we enable re-use and composition of your exiting tools! This is powered by a powerful task resolution system, an extremely reliable execution technique and an advanced recursive queueing system. At the same time we provide best-in-game error handling, features that are not possible in other tools and an all-round better developer experience.

1. Install $ npm install -g crossbow
2. Compose existing tools, plugins and scripts
crossbow.yml cbfile.js crossbow.js
const cb = require('crossbow');
const vfs = require('vinyl-fs');

cb.task('build', ['scripts', 'styles']);

cb.task('scripts', [
    '@sh rm -rf ./dist',
    '@npm webpack'
]);

cb.task('styles', function () {
    return vfs.src('./app/scss/core.scss')
        .pipe(sass())
        .pipe(vfs.dest('./dist/css'));
});

tasks:

  build:
    - styles
    - scripts

  scripts:
    - '@sh rm -rf ./dist'
    - '@npm webpack --optimize'

  styles:
    tasks:  tasks/sass-task.js
    options:
      input: 'app/scss'
      output: 'dist/css'

module.exports = {
    tasks: {
        build: [
            'styles',
            'scripts'
        ],
        scripts: [
            '@sh rm -rf ./dist',
            '@npm webpack'
        ],
        styles: {
            tasks: 'tasks/sass-task.js',
            options: {
                input: 'app/scss',
                output: 'dist/css'
            }
        }
    }
};
3. run $ cb build
